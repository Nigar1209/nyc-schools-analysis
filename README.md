# 🗽 NYC Schools Analysis

This project explores real-world data from **New York City's public school system** to uncover insights related to **school incidents, SAT scores, and differences across NYC boroughs**.  
The analysis combines **data exploration, SQL querying, and database modeling** to better understand patterns in education data.

---

## Overview

The project is organized into **four main analytical components**, each focusing on a different dataset and aspect of NYC schools.

---

## 🏫 Incident Analysis
This section analyzes **school safety reports from NYC public schools (2013–2016)**.

The goal is to explore patterns in **criminal and non-criminal incidents** and how they vary across boroughs.

Key focus areas include:
- Distribution of incident types
- Differences between boroughs
- Trends in school safety reports

---

## 📋 School Directory Exploration
This section analyzes the **NYC school directory dataset** to better understand the structure of public schools across the city.

The analysis focuses on:
- School distribution across boroughs
- Types of schools available
- Differences in school characteristics and program offerings

---

## ⛁ Database Queries
This component explores the **database schema and relationships between datasets** using SQL.

The notebook includes queries analyzing:
- School distribution across boroughs
- Percentage of **English Language Learners (ELL)**
- Indicators related to **support for special needs students**

This part demonstrates how **relational data can be queried to extract insights across multiple tables**.

---

## ➡️ Database Population
This section implements a **basic data pipeline** that populates the database with NYC school data.

The notebook:
- Cleans and prepares the school directory dataset
- Ensures **data integrity and consistency**
- Loads the processed data into a **PostgreSQL database**

This step enables further **SQL-based analysis and scalable data exploration**.

---

## Tools

The project uses a combination of data analysis and database technologies:

- **Python (Jupyter Notebooks)**
- **Pandas, NumPy, Matplotlib** for data analysis
- **Google Sheets** for initial data exploration
- **PostgreSQL** for database storage and querying
